# SkillSwap System

SkillSwap is a simple web platform where users can share and exchange skills with each other. The idea is to make learning more interactive by allowing people to connect, send requests, and help each other grow.

## Features
- User signup and login
- Add and browse skills
- Send and accept requests
- Real-time chat between users
- Notifications (popup and sidebar)
- Option to share media like images or videos

## Tech Used
- HTML, CSS, JavaScript
- PHP
- MySQL

## How to Run
1. Install XAMPP  
2. Move the project folder to `htdocs`  
3. Start Apache and MySQL  
4. Open phpMyAdmin and import `skillswap.sql`  
5. Open in browser:  
   `http://localhost/skillswap`
